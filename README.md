# Sayantan's Blog

A modern blog built with Astro 5, featuring:

- 📝 MDX support for rich content
- 📧 Newsletter with Resend integration
- 🎨 Tailwind CSS for styling
- 📱 Responsive design
- 🌙 Dark mode support
- 📊 RSS feed
- 🚀 Auto-deployment

## Development

```bash
npm install
npm run dev
```

## Deployment

Push to main branch for automatic deployment.

## Environment Variables

- `RESEND_API_KEY`
- `RESEND_AUDIENCE_ID`
- `SITE_URL`
